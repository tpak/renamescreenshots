📷 Screenshot Renamer

User Guide

What It Does

Screenshot Renamer is a lightweight macOS menu bar app that automatically renames your screenshots from the default 12-hour time format to 24-hour (military) time. This makes screenshots sort chronologically in Finder, regardless of whether they were taken in the morning or afternoon.

Screenshot 2026-02-24 at 2.30.45 PM.png
→ Screenshot 2026-02-24 at 14.30.45.png

The app watches your screenshot folder in real time and renames new files as they appear. Screenshots already in 24-hour format are left untouched.

Getting Started

Download & Install

  1. Download the latest DMG or ZIP from the Releases page.
  2. If you downloaded the DMG, open it and drag Screenshot Renamer to your Applications folder. If you downloaded the ZIP, unzip it and move the app to Applications.
  3. Double-click the app to launch it. Because the app is not notarized, macOS Gatekeeper will block it on first launch.

Bypassing Gatekeeper

Since the app is distributed outside the Mac App Store without notarization, macOS will show a warning the first time you open it. To allow it:

  1. Right-click (or Control-click) on Screenshot Renamer.app in Finder.
  2. Select Open from the context menu.
  3. Click Open in the confirmation dialog.

You only need to do this once. After that, the app opens normally.

First Launch

When the app starts, a small camera icon (📷) appears in your menu bar. The file watcher starts automatically, monitoring your current screenshot folder for new files to rename.

Menu Bar

Click the camera icon in the menu bar to access the app. The menu shows:

  • Stop/Start Watcher — Toggle the file watcher on or off.
  • Quick Rename... (⌘R) — Scan and rename existing screenshots in the current folder.
  • Settings... (⌘,) — Open the settings window.
  • User Guide — Open this page in your browser.

Below the actions, the menu displays your current configuration at a glance: save location, prefix, format, and a summary of active options.

Settings

The settings window is organized into three sections: Screenshot Options, App Options, and Debug.

Screenshot Options

Save Location — Shows where macOS saves screenshots. You can type a path directly or click Choose... to pick a folder. Changing this updates the macOS system setting.

Prefix — The filename prefix macOS uses when saving screenshots. The default is Screenshot. You can change it to any non-empty string (for example, Screen Shot or Capture). The renamer automatically adapts its matching pattern to your chosen prefix.

  • Show thumbnail preview after capture — Controls the floating thumbnail that appears briefly in the corner after taking a screenshot.
  • Include mouse pointer in screenshots — When enabled, the cursor is visible in captured images.
  • Disable window shadow — Removes the drop shadow from window screenshots, useful for cleaner images.
  • Include date in filename — Toggles whether the date and time appear in the screenshot filename.

Format — Choose the image format for new screenshots: PNG (default), JPG, PDF, or TIFF.

Capture Delay — Set a timer before each screenshot is taken: None (instant), 5 Seconds, or 10 Seconds. Useful when you need time to arrange windows before capturing.

App Options

  • Launch at login — Starts Screenshot Renamer automatically when you log in to your Mac.
  • Automatically check for updates — Periodically checks for new versions. Choose a frequency: Daily, Weekly, or Monthly.
  • Check Now — Immediately check for a new version without waiting for the next automatic check.
  • Automatically download and install updates — When enabled, updates are downloaded and installed in the background without prompting.

Debug

Enable debug logging to write detailed activity to a log file. Use the Open button to view the log in Finder, or Clear to empty it. You can change the log file location with Set Location....

Reset to Defaults

Restores all screenshot settings to macOS defaults: location to Desktop, prefix to "Screenshot", PNG format, no delay, and standard options.

Taking Screenshots on macOS

macOS provides several built-in keyboard shortcuts for capturing your screen:

  • ⌘ ⇧ 3 — Capture the entire screen and save it as a file.
  • ⌘ ⇧ 4 — Enter selection mode. Click and drag to capture a rectangular area of the screen.
  • ⌘ ⇧ 4 then Space — After pressing the shortcut, hit Space to switch to window capture mode. Click any window to capture it with its shadow (unless shadow is disabled in settings).
  • ⌘ ⇧ 5 — Open the Screenshot toolbar, which lets you choose between capturing the entire screen, a selected window, or a selected portion. The toolbar also provides options for recording your screen.
  • Hold ⌃ Control with any of the above shortcuts to copy the screenshot to your clipboard instead of saving it as a file.
  • Press Esc at any time to cancel a screenshot in progress.

Troubleshooting

Screenshots are not being renamed

  • Make sure the watcher is running — the menu should show Stop Watcher (meaning it is active). If it says Start Watcher, click it to start.
  • Verify the save location in Settings matches where your screenshots are actually being saved.
  • Check that Include date in filename is enabled — without the date/time in the filename, there is nothing for the renamer to convert.

The app does not appear in the menu bar

  • Screenshot Renamer runs as a menu bar app with no Dock icon. Look for the camera icon in the right side of your menu bar.
  • If the menu bar is crowded, the icon may be hidden. Try closing other menu bar apps to make room.

Quick Rename does not find any files

Quick Rename scans the current screenshot folder for files matching the 12-hour pattern. If all files are already in 24-hour format or use a different naming pattern, there is nothing to rename.

Using the debug log

If something isn't working as expected, enable debug logging in Settings. The log records every file event, rename operation, and error. You can share the log file when reporting an issue on GitHub.
